Intelligent Virtual Personal Assistant Comprising Disease Forecasting

  • Tech Stack: Python GUI, Pandas, Tensorflow, HTML, CSS, Scikit Learn
  • Github URL: Project Link

The advent of virtual personal assistants marked a significant milestone in the realm of computer science, revolutionizing user interactions through speech-enabled operations. This project led a team of four to harness this innovation by integrating disease prediction capabilities into a Windows Enterprise virtual personal assistant. Employing Natural Language Processing (NLP) and Automatic Speech Recognition (ASR), the project elevated user experience and efficiency by providing health insights. This initiative resonates with the data-driven era, illustrating the potential of virtual assistants across industries such as healthcare, automotive, marketing, and education. The project's outcomes included features like hands-free system commands, file searches, and voice-based note-taking.
